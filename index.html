What will I be learning?

Web Dev. Fundamentals (HTML, CSS, JavaScript)

Learn the fundamentals needed to build websites for the internet. Covering topics beginning with HTML and CSS all the way to advanced JavaScript.

Frontend Development (React JS)

Develop the skills you will need to analyze code, design and debugging applications while maintaining a seamless user experience.

Backend Development (Ruby and Ruby on Rails)

Become skilled in writing code that connects the web to a database, manages user connections and powers the web application as a whole.
